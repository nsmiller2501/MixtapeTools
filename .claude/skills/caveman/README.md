# /caveman

A communication mode that strips all filler from Claude's responses — no articles, no pleasantries, no hedging — while preserving complete technical accuracy. Cuts token usage by roughly 75%. Triggered by saying "caveman mode", "less tokens", or `/caveman`; turned off with "normal mode" or "stop caveman". Stays active until explicitly dismissed.
