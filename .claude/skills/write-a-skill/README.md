# /write-a-skill

A meta-skill for creating new Claude Code skills. Claude interviews you about the task you want to automate, drafts the `SKILL.md` instruction file (plus any supporting reference files or scripts), presents it for review, and iterates until you're satisfied. Follows the skill structure conventions used throughout this repo: a concise description that triggers correctly, a `SKILL.md` under 100 lines, and supplementary files split out when needed.
