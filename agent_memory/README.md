# Agent Memory

Tracked durable memory for this skill-development repo.

Use this folder for decisions and rejected/deferred ideas that should survive branch deletion, rollbacks, and ignored `progress_logs/` churn.

Use `progress_logs/` for session continuity: task lists, command history, branch-local notes, and work-in-progress paper trails.

Current files:

- `key_decisions.md` — durable architectural decisions.
- `deferred_ideas.md` — postponed or rejected ideas that should remain visible.
