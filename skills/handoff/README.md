# /handoff

Compacts the current conversation into a handoff document so a fresh Claude session can pick up exactly where you left off. Rather than duplicating artifacts (plans, diffs, ADRs, open issues), the document references them by path or URL. Accepts an optional argument describing what the next session will focus on, which narrows the summary accordingly.

Useful when a session is running long, when you want to switch context, or when you're handing work off to another person or agent.
